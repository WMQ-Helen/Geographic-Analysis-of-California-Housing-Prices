# Overview

This project uses QGIS to visualize the spatial distribution of housing prices in California. Using geographic coordinates from the California Housing dataset, the project maps median house values to explore how housing prices vary across different regions.

# Dataset

The `housing.csv` file was downloaded from the California Housing Data (1990) dataset published by Harry Wang on Kaggle.

The dataset contains 20,640 observations, with each record representing a California census block. Variables include:

- Longitude
- Latitude
- Median House Value
- Median Income
- Housing Median Age
- Population
- Households
- Total Rooms
- Total Bedrooms
- Ocean Proximity

# Software

- QGIS
- OpenStreetMap (XYZ Tiles)
- California Housing Dataset (Kaggle)

Coordinate Reference System (CRS):
- EPSG:4326 (WGS84)

# Results

The map reveals several noticeable spatial patterns.

- The San Francisco Bay Area contains the highest concentration of high-value housing.
- Southern California also includes multiple areas with relatively high housing values.
- Inland regions generally exhibit lower housing prices than coastal regions.
- The dark red highlights clear geographic differences in housing values across California.
